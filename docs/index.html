政策纲要
《教育强国建设规划纲要（2024—2035年）》发布
中共中央、国务院印发《教育强国建设规划纲要（2024—2035年）》，旨在加快建设中国特色社会主义教育强国。文件明确了中长期教育发展的战略目标与核心任务，是未来十余年教育工作的顶层设计与行动纲领。

年度重点
2026年全国教育工作会议明确核心方向
2026年全国教育工作会议近期举行，明确了今年教育改革发展的核心方向。会议提出推进健康学校建设，加快普及全学段的人工智能通识教育，并强调要更好应对学龄人口变化，勾勒出教育高质量发展新蓝图。
来源：edu.cctv.com2026-01-26阅读原文 →

教育部基础教育司负责人就《关于开展基础教育规范管理巩固年行动的通知》答记者问。此举旨在进一步规范基础教育办学行为，巩固管理成效，营造良好教育生态，推动基础教育持续健康发展。

德英乐视角
