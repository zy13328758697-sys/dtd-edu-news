教育部开展基础教育规范管理巩固年行动，发布2026版负面清单
教育部办公厅于2026年3月27日发布通知，启动基础教育规范管理巩固年行动。通知要求各地加强规范管理基本要求和负面清单禁止行为的宣传解读，确保落实到每所学校。同时，强调会同有关部门加强网络生态治理，对散布不实谣言、制造“教育焦虑”、杜撰虚假教育政策等行为依法处置，营造清朗网络空间。附件中发布了《基础教育规范管理负面清单(2026版)》。

心理健康
教育部出台中小学生心理健康十条措施，聚焦考试焦虑与睡眠
2026年全国教育工作会议期间，教育部于10月9日出台《进一步加强中小学生心理健康工作十条措施》。措施重点包括缓解学生考试开学焦虑、落实“体育每天2小时”、保障充足睡眠时间、培养健康用网习惯等。该政策旨在系统性地提升中小学生心理健康水平，减轻学业压力带来的负面影响。

IB、AP、A-Level三大国际课程体系全解析：适合的才是最好的
