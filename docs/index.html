政策与监管动态
规范管理
教育部启动基础教育规范管理巩固年行动，发布2026版负面清单
教育部办公厅于2026年3月27日发布通知，开展基础教育规范管理巩固年行动。通知要求各地加强规范管理基本要求和负面清单禁止行为的宣传解读，确保落实到每所学校、校长教师人人知晓。同时，将强化正面宣传引领，会同有关部门加强网络生态治理，对散布不实谣言、制造“教育焦虑”、杜撰虚假教育政策等行为依法依规处置，营造清朗网络空间。
来源：中国教育在线2026-03-27阅读原文 →
德英乐视角
德英乐需对照2026版负面清单全面自查，确保办学行为合规，同时加强家校沟通，避免因信息不实引发焦虑。
招生规范
教育部部署义务教育阳光招生专项行动，严格规范民办学校招生
教育部发布《关于开展义务教育阳光招生专项行动（2025）的通知》，要求严格规范民办学校招生行为，对民办学校开展全面排查，对有停办风险的学校，属地教育行政部门需提前预警并做好学生安置预案。同时，全面落实“两为主、两纳入、以居住证为主要依据”的随迁子女义务教育入学政策，加强流动儿童入学保障。
来源：中国政府网2025-03-18阅读原文 →
德英乐视角
德英乐旗下民办学校需提前排查办学风险，确保招生流程透明合规，并关注随迁子女入学政策对生源结构的影响。
教师减负
教育部出台措施减轻中小学教师非教育教学负担
教育部教师工作司负责人就《关于进一步减轻中小学教师非教育教学负担若干措施的通知》答记者问，明确将清理规范各类督查检查评比考核、社会事务进校园等事项，切实减轻教师负担，让教师回归教书育人主业。该政策旨在保障教师专注教学，提升教育教学质量。
来源：教育部2025-11-10阅读原文 →

德英乐应优化教师管理流程，减少非教学事务干扰，提升教师幸福感与教学专注度，从而保障双语课程质量。
行业趋势与市场
AI教育
2026年全球AI教育元年：超70%国家已出台专项政策
中国成人教育协会专家岑建君指出，2026年被全球多国确立为AI教育全面启动元年。中国在全国教育工作会议上正式将2026年定为AI教育全面启动元年，系统推进人工智能进课程、进教学、进评价。美国、韩国、芬兰等也发布国家级AI教育战略。据统计，全球超过70%的国家和地区已出台专项政策，推动AI教育从试点走向规模化。
来源：中国成人教育协会2026-03-10阅读原文 →

教育行业2026年度投资策略：政策改善+AI赋能，业绩与估值共振
教育行业2026年投资策略报告指出，政策改善与鼓励生育政策将释放教育需求，教育行业呈现“剩者为王”特点。同时，“AI+教育”商业化加速，成为行业核心增长点。报告认为，教育科技人才一体统筹，教育强国与人工智能双轮驱动行业新发展，教育上市公司成为此轮市场复苏的主要受益者。
