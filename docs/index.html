政策与监管动态
政策纲领
中共中央、国务院印发《教育强国建设规划纲要（2024—2035年）》
《纲要》是未来十余年教育发展的顶层设计，明确提出要“深化教育综合改革，激发教育发展活力”，并再次强调“引导规范民办教育发展”。这为包括民办双语教育在内的各类教育主体指明了宏观政策导向和发展方向，标志着民办教育进入鼓励支持与规范发展并重的新阶段。
来源：中华人民共和国教育部2026-04-21阅读原文 →
德英乐视角
《纲要》为德英乐这类优质民办双语学校提供了长期稳定的政策预期。我们应积极解读，在规范框架下，持续聚焦教育质量与特色发展，将政策引导转化为办学优势。
民办规范
教育部等八部门关于规范公办学校举办或参与举办民办义务教育学校的通知
该通知旨在进一步规范“公参民”学校的办学行为，厘清公办与民办学校的界限，推动义务教育领域更加公平、规范的发展。这是对民办教育分类管理、规范发展的具体政策落实，有助于净化市场环境，让真正有特色、高质量的民办学校获得更清晰的发展空间。
来源：平昌县人民政府2025-07-01阅读原文 →
德英乐视角
政策持续规范义务教育阶段办学，有利于拥有独立品牌和完整体系的德英乐学校。我们始终坚持合规办学，专注自身课程与师资建设，在规范市场中凸显一贯制双语教育的独特价值。
AI+教育
教育部等五部门印发《“人工智能+教育”行动计划》
该行动计划是国家层面推动AI与教育深度融合的纲领性文件，旨在通过政策引导，加速人工智能技术在教育场景的应用、创新与规范发展。文件明确了发展目标、重点任务和保障措施，为教育科技行业提供了明确的政策红利和发展路径。
来源：教育在线2026-04-10阅读原文 →
德英乐视角
德英乐需密切关注并积极对接国家“AI+教育”战略。可探索将AI工具融入双语教学、个性化学习路径规划及教学管理，提升教育效率与体验，同时注重教育本质与数据隐私保护。
行业趋势与市场
AI教育趋势
2025教育AI复盘与2026市场展望：从技术落地到智能体爆发
报告指出，2025年中国教育AI市场规模已突破1125亿元。展望2026年，行业增长点将从基础技术应用转向智能体（Agent）、端侧隐私计算以及非认知能力培养三大新方向。AI正从辅助工具演变为能够深度参与教学过程、提供个性化服务的“智能伙伴”。
来源：OpenAxo2026-04-08阅读原文 →
德英乐视角
德英乐应前瞻性布局教育智能体在双语教学、升学规划、学生成长陪伴等场景的应用研究。同时，将AI作为培养学生批判性思维、创造力等非认知能力的赋能工具，而非替代品。
民办教育调研
韦伯咨询：2025年中国民办教育行业专题调研与深度分析报告
该深度报告全面分析了“双减”后民办教育行业的新格局。报告指出，行业监管逻辑已切换，市场格局趋于清晰稳定。未来，民办学校的发展将更依赖于特色化、高质量的内涵式增长，以及对政策红利的精准把握和合规运营能力。
来源：搜狐2025-01-01阅读原文 →
德英乐视角
报告印证了德英乐坚持“高质量、特色化”发展路径的正确性。在行业格局重塑期，我们更应夯实双语融合课程体系，强化海外升学优势，以卓越的教育成果赢得市场认可。
AI课程落地
教育部AI必修课2026秋季全国落地：家长完整解读
自2026年秋季学期起，全国中小学将从小学三年级到高中三年级统一开设人工智能必修课，每周至少1课时。这标志着AI素养教育正式纳入国家基础教育体系，成为学生的必备技能。政策将驱动学校课程改革、师资培训及教学资源建设的巨大需求。
来源：Kids AI Tools2026-04-01阅读原文 →
德英乐视角
作为拥有IB、CAIE等国际认证的学校，德英乐可率先将国家AI课程要求与我们的国际课程（如IB的ATL技能）进行创新融合，设计出兼具前沿性与实践性的校本AI课程，形成新的教学特色。
国际教育视野
国际课程
国际学校课程体系多元化：IB、A-level、AP及双语课程并行
当前全球及中国的国际学校普遍提供多元化的课程选择，包括国际文凭（IB）、英国A-Level、美国AP课程以及各类双语课程。这种多元体系允许学生根据个人学术特长、兴趣及目标留学国家，选择最适合自己的升学路径，体现了国际教育的灵活性和个性化导向。
来源：羊城晚报2025-05-28阅读原文 →
德英乐视角
德英乐高中阶段聚焦海外升学，已构建了成熟的课程支持体系。我们需持续优化课程顾问服务，精准指导学生结合自身优势选择IB等课程，实现与全球顶尖大学的顺畅对接。
全球实践
葡萄牙国际学校提供双语课程，满足外籍家庭多元化需求
以葡萄牙的国际学校为例，为满足全球外籍人士家庭的需求，许多学校提供英国国家课程、美国课程、国际文凭（IB）以及专门的双语课程。这反映了全球范围内，高质量的双语教育已成为国际学校吸引多元化生源、培养全球公民的重要标配。
来源：Anchorless2025-01-01阅读原文 →
德英乐视角
这印证了德英乐从小学到高中一贯制双语道路的全球普适性与前瞻性。我们应持续深耕“双语双文化”融合教育，打造区别于单纯出国预备班的、根植中国面向世界的教育品牌。
国际生管理
教育部等三部令第42号《学校招收和培养国际学生管理办法》
该管理办法规范了各级各类学校招收、培养、管理国际学生的行为，涉及入学条件、教学管理、校内管理、社会管理以及监督管理等方面。随着教育国际化深入，该办法为学校开展国际学生教育提供了明确的法规依据和操作规范。
来源：中国传媒大学国际学院2025-04-24阅读原文 →
德英乐视角
德英乐旗下学校虽主要服务中国学生，但该办法为我们未来可能拓展的短期国际交流生项目、招收外籍学生等国际化举措提供了重要的政策合规指南，需纳入学校管理体系的考量。
教育深度观察
行业转型
政策清晰、AI平权，2026年教育行业有哪些新机会？
分析认为，2025年是“双减”后教育行业格局走向清晰与稳定的一年。监管逻辑完成底层切换，AI技术实现全面渗透。进入2026年，行业新机会在于：在明确的政策框架下，利用AI等技术实现教育服务的个性化、普惠化与效率提升，以及探索素质教育、教育出海等新赛道。
来源：36氪2026-04-01阅读原文 →
德英乐视角
德英乐身处行业转型前沿。我们的机会在于：利用AI赋能个性化双语学习，巩固K-12一贯制教育生态的护城河，并凭借成熟的国际课程经验，探索教育理念与模式的对外交流与合作。
