AI教育
教育部等五部门印发《“人工智能+教育”行动计划》

规范管理
教育部开展基础教育规范管理巩固年行动

减负政策
教育部解读减轻中小学教师非教育教学负担措施
2025年11月，教育部教师工作司就《关于进一步减轻中小学教师非教育教学负担若干措施的通知》答记者问，明确要求精简各类检查评比、社会事务进校园等非教学任务，保障教师专注教育教学。该政策旨在提升教师职业幸福感与教学质量，对民办双语学校师资管理提出更高要求。

国际课程选择焦虑：家长如何避免信息陷阱？
