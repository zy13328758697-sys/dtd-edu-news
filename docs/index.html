政策与监管动态
民办限令
北京发布民办在校生5%限令，民办教育“瘦身”提速
北京出台民办义务教育在校生占比不超过5%的限令，标志着民办教育“瘦身”政策加速落地。该政策要求民办学校明确办学定位、彰显办学特色，提供多元化教育供给。在“更名潮”背后，民办义务教育学校正发生结构性转向，从规模扩张转向质量提升，行业洗牌加剧。

资金监管
义务教育阶段民办学校不得举借债务，加强资金监管
最新政策明确义务教育阶段民办学校不得举借债务，并要求加强民办学校资金监管。此举旨在防范财务风险，确保学校运营稳健。民办学校需严格规范财务行为，避免因债务问题影响正常教学秩序，同时需提升资金使用透明度。

2026：全球AI教育元年，中国系统部署人工智能进课程
中国在全国教育工作会议上正式将2026年确立为AI教育全面启动元年，全面推进人工智能进课程、进教学、进评价。美国、韩国、芬兰等也先后发布国家级AI教育战略，全球超过70%的国家和地区已出台专项政策，将2026年定为AI教育从试点走向规模化的核心起点。

国际学校“催生”史：资本与教育互搏二十年

AP/IB/A-Level三大国际课程怎么选？99%国际生家长最关心的问题
目前国内主流的国际课程体系为A-Level、AP和IB，各具特点：A-Level适合偏科学生，AP适合冲刺美国名校，IB注重全人教育。2024年《中国留学白皮书》显示，选择国际课程的高中生已突破12万人，三大体系占据95%以上份额。选课需结合学生兴趣、升学方向及学校资源综合考量。

五大国际课程盘点：A-Level、AP、IB、OSSD、DSE最新对比
随着教育国际化发展，A-Level、AP、IB、OSSD和DSE五大国际课程成为热门选择。OSSD侧重过程性评估，DSE适合港澳升学。课程选择需考虑留学认可度、学生能力及家庭预算。2025年数据显示，IB课程全球认可度最高，但难度也最大。
